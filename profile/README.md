# 🌿 GratiaOS

> Gratia is a resilient environment for memory and presence.
> A canonical monorepo where systems stay local-first, private, and coherent.

GratiaOS is the active source of truth for the Gratia stack. It is designed for offline-resilient development, shared package coherence, and low dependency on external platforms.

## 🏛️ Architecture

- [gratiaos/apps/m3](https://github.com/GratiaOS/gratiaos/tree/main/apps/m3) — local-first memory and knowledge system
- [gratiaos/apps/firegate](https://github.com/GratiaOS/gratiaos/tree/main/apps/firegate) — companion interface and household-facing layer
- [gratiaos/apps/amigo](https://github.com/GratiaOS/gratiaos/tree/main/apps/amigo) — quiet link transport with note and room flow
- [gratiaos/services/firecircle](https://github.com/GratiaOS/gratiaos/tree/main/services/firecircle) — small-group presence interface
- [gratiaos/packages/garden-core](https://github.com/GratiaOS/gratiaos/tree/main/packages/garden-core) — shared tokens, primitives, and presence kernel
